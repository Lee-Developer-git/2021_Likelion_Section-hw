## selector 차이

자식선택자
-> 자식(바로 아래 태크) 중 선택자와 일치하는 요소 선택
```css
article > p {
    color: red;
}
```

후손선택자
-> 후손들 중 선택자와 일치하는 요소 선택
```css
article p {
    color: red;
}
```
